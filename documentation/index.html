Using The Big Day - Responsive One-Page Wedding Template

File structure

Folder and files you find in the template are explained below.

Folders

  • html - main folder, contains post and pre wedding templates.
    • post-wedding - folder for the post wedding template.
    • pre-wedding - folder for the pre wedding template.
      • css - contains all the css file used in the template.
        • fancybox - contains Fancybox related files.
      • images - contains all the images.
      • js - contains all the javascript files.
  • psd - contains all the PSDs files.
  • documentation - contains help file.

HTML files

  • index.html - contains the full HTML code for each template.
  • send-mail.php - only for pre wedding template, contains the PHP script for sending emails.

CSS files

  • bootstrap.css - grid structure based on Bootstrap Framework.
  • bootstrap-responsive.min.css - grid structure for mobile viewport.
  • flexslider.css - styles for Flexslider Slider.
  • fancybox/jquery.fancybox.css - styles for Fancybox.
  • style.css - main custom styles.
  • style.responsive.css - main custom styles for mobile viewport.

JavaScript files

  • jquery.min.js - main jQuery library.
  • jquery.fancybox.pack.js - main javascript for Fancybox.
  • jquery.easing.pack.js - jquery advanced easing options.
  • jquery.mousewheel.pack.js - adds mouse wheel support for Fancybox.
  • jquery.flexslider.min.js - main javascript for Flexslider Slider.
  • jquery.validate.min.js - main javascript for contact form validation.
  • big-day.js - custom javascript.

Installation

Files are written in simple HTML/CSS/JavaScript, so you do not need an internet connection in order to display or edit the template, you can view this template in any web browser from your computer. Except for the contact form, as it requires a PHP script and then must run on a server.
It's important to understand that you must have an online server in order to use this landing page. The server can be purchased at many different places around the internet. It's a simple thing: You need to put the files that you purchased somewhere so others can find them online.

In order to change the template, please refer to the Customizations section. Make sure you also change the author and description meta tags with your own.

							<meta name="description" content="Description.">
							<meta name="author" content="Your Name Here">

You can find this block of code at line 10 of each HTML file.

Upload to your server

  1. Select /html folder and then choose the landing page template that you want. As you can see, there are several files/folder.
  2. Start your FTP client (I recommend FileZilla, which is free) and login to yout server.
  3. Locate the domain folder you wish to upload your template files to on your server. You are going to want to click and drag every file and folder that you have inside the template folder into your FTP client.
  4. Once the files are done uploading, you can access your website via browser.
